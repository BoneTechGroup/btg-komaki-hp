## 決まっていること

| 項目 | 内容 |
|---|---|
| 作業場所 | `work/hp-komaki`（seo-automation とは混ぜない） |
| キャラクター | **プロのイラストレーターに依頼**予定 |
| 私（Claude）の役割 | サイトの土台（デザイン・構成・キャラをはめ込む枠）を先に作る |
| 公開 | HPを先に作り、**ドメイン取得・公開は後から**でOK |
| プラットフォーム | 凝ったデザイン＋ブログ集客を両立 → WordPressカスタム or 静的サイト＋ブログ（要検討） |

### プラットフォームの考え方（メモ）
- 「凝った動くデザイン」と「ブログ量産」は両立できる。
- WordPress（カスタムテーマ）なら1つで両方カバー可能。
- 静的HTMLでも作れるが、ブログ自動投稿はWordPress向けスクリプトを流用するのがラク。

## 次にやること（再開時の手順）

1. **院の基本情報をもらう**（下記テンプレート）
2. それをもとに **サイト設計図（ページ構成＋キャラ登場プラン）** を作成
3. **トップページを1枚試作**してローカルで確認
4. OKなら全体を作り込む → 後でドメイン取得・公開
