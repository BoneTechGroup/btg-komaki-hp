# 表現・NGワードルール（接骨院の広告規制／医師法・柔整師法）

接骨院（柔道整復師）のサイトでは、医療機関（病院・医師）を想起させる表現が使えない。
医師法・柔道整復師法・広告ガイドラインに抵触しないよう、以下のワードは**必ず言い換える**。

## NGワード → 言い換え

| NG（使わない） | OK（言い換え） |
|---|---|
| 診療 / 診察 | 施術 |
| 治療 | 施術（または「ケア」「おケガのケア」） |
| 問診 | カウンセリング / ヒアリング / お体の状態の確認 |
| 診断 | 検査 / お体の状態の確認 |
| 患者（が主語の断定的表現） | 患者さん / 来院者（文脈で調整） |
| 〇〇を治す・完治 | 改善のサポート / 回復のサポート |
| カルテ | 施術記録 |

## 例外
- **資格・肩書きの固有名称**は、その名称の一部に「治療」等が含まれても、名称としてそのまま可。
  - 例:「交通事故治療専門アドバイザー」（民間資格の名称）
- 保険・法律上の用語で正式名称のものは文脈により可だが、可能なら平易な言い換えを優先。
  - 例:「総治療日数」→ サイト上では「通院期間（日数）」と表記。

## 運用
- 新しく文章を追加・編集するときは、必ずこの表を確認してNGワードを避ける。
- 迷ったら「施術」「カウンセリング」「検査」「サポート」に寄せる。
